A) Installation - top

  1. Upload plugin to the /wp-content/plugins/ directory or use 'Add New Plugin'.
  2. Activate the plugin through the 'Plugins' menu in WordPress

B) Settings - top

you will find menu 'My Rules' in admin menu and can manage your access level for all roles as well as post types including custom post types and guests(not logged in users).You can also manage access level for single post/page/attachment or custom post type.
