== Description ==

This plugin allow to manage user roles with post types very effectively.You can restrict your visitor/users to access any post type as well as an individual post type like single post, page, attachment etc.


== Installation ==

This section describes how to install the plugin and get it working.


1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation the plugin you guys can find in Menu with the name 'My Rules'.
